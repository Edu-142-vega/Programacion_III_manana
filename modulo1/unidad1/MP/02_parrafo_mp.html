Un registro formal de una solicitud de ayuda o de un problema técnico que un usuario crea para un equipo de soporte

Un ticket se genera cuando un cliente o usuario envía una solicitud o problema, se atiende asignando el ticket a un agente, resolviendo el problema mediante comunicación y acciones, se cierra una vez que la solicitud ha sido resuelta, notificando al cliente y archivando la información
